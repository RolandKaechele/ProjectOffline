Task Creation & Editing

Add, edit and organise tasks in the task sheet — from simple one-liners to structured summary hierarchies.

Adding Tasks

  • Right-click any row → Insert Task
  • Press Insert while the task list has focus
  • Toolbar Add Task button

New tasks are inserted above the selected row at the same outline level.

Editing Tasks

Double-click any row to open the Task Information dialog. You can also double-click a task bar in the Gantt chart.

FieldNotes
NameTask display name.
DurationEnter as Nd (days), Nw (weeks), or Nh (hours).
% CompleteProgress 0–100 (or more for tracking over-runs).
Start (constraint)Optional – leave blank to let the schedule compute it.
NotesFree-text; shown as a 📄 icon in the indicator column.
ResourcesAssign from the project resource list. Each assignment row shows the resource name and a ✕ delete button to remove that specific assignment.

Summary Tasks & Indentation

Tasks become summary tasks when they have children. Use the Indent and Outdent toolbar buttons (or Alt+Shift+Right / Alt+Shift+Left) to move a task one outline level deeper or shallower. Summary task bars span all their children and cannot be edited directly — they reflect the aggregate of child dates.

Milestones

Set Duration to 0d to make a task a milestone. Milestones appear as a diamond ◆ in the indicator column and as a diamond shape on the Gantt chart. They can optionally be pinned to the Timeline strip.

Deleting Tasks

Select one or more rows and press Delete, or right-click → Delete Task. Deleting a summary task also deletes all its children. Use Undo (Ctrl+Z) to reverse.
