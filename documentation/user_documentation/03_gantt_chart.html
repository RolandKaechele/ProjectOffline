Gantt Chart

Understand and navigate the Gantt chart panel — task bars, zoom levels, critical path and today indicator.

Home› Gantt Chart

Reading the Chart

The Gantt chart occupies the right half of the Gantt Chart tab. Each task has a horizontal bar whose position represents its start date and whose width represents its duration. The two-row header shows months (top band) and day numbers (bottom band).

Visual elementMeaning
Blue barNormal in-progress task
Red barCritical path task (zero float, project finish depends on it)
Diamond ◆Milestone (0-day duration)
Dark strip below barBaseline (when a baseline is set; shows planned dates)
Green vertical lineToday indicator — bold line with downward triangle
Grey columnsSaturdays and public holidays
Blue-tint columnSundays (can be hidden via Options)

Zooming

Use the zoom slider in the toolbar or the Zoom In / Zoom Out buttons (+ / -). There are four display modes:

Pixel width / dayModeHeader bottom row
4–6 pxMonth overviewWeek numbers
7–13 pxWeek overviewWeek numbers + day abbreviations
14–59 pxDay detail (default ~22 px)Day numbers
60–80 pxHourlyHour slots from the project calendar

Navigation Buttons

Above the calendar header:

ButtonAction
◀◀ FirstScroll to 1 week before the first task's start
◀ TodayScroll to 1 week before today
Last ▶▶Scroll to 1 week before the last task's finish

Critical Path

Enable via Options → Show Critical Path or the toolbar toggle. Tasks on the critical path are redrawn in red. A task is critical when its total float is zero (or — if Zero Float = Critical is active — exactly zero float). The Gantt chart and Task Sheet both reflect the current state.

Opening Task Information

Double-click a task row in the task panel (the left-hand grid showing task names) to open the Task Information dialog. The dialog receives the current CPM critical-path data so the Schedule tab displays the correct critical status and float values without needing to switch to the CPM Results panel first.

Drag-Moving Tasks

Click and drag a task bar horizontally to reschedule it. A ghost bar shows the proposed position while you drag. Releasing the mouse updates the task start date and emits a task_moved event so downstream logic can recalculate dependencies. Summary task bars cannot be dragged directly.

Sundays Toggle Sunday column visibility via Options → Show Sundays. When hidden, the Gantt skips Sunday entirely and the day-pixel mapping compensates so bar widths remain correct.
