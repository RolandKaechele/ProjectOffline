Overview & Interface

A guided tour of the Project Offline window: tabs, toolbar, and how to open your first project.

Home› Overview & Interface

Opening a Project

Start Project Offline and use File → Open… (or the toolbar button) to open a project file. Supported formats:

ExtensionFormat
.mppMicrosoft Project binary (read-only source; saves as XML)
.xml / .mspdiMicrosoft Project XML (native save format)
.mptMicrosoft Project template
.mpxMicrosoft Project Exchange

Main Window Layout

Tabs

TabWhat it shows
Gantt ChartTask sheet (left) + Gantt bars (right). Primary editing area.
ResourcesResource sheet — names, types, max units, pay rates.
Team PlannerResource-row timeline with drag-and-drop task assignment.
Resource UsagePer-resource hour breakdown at Monthly / Weekly / Daily / Hourly zoom.
BaselineSide-by-side comparison of planned vs. current dates with variance ? columns.
DependenciesTable of all predecessor relationships (FS / FF / SS / SF).
Task SheetFlat editable task grid without the Gantt chart panel.

Timeline Strip

The compact strip above the tab bar shows the full project span, pinned task bars, and milestone diamonds on a condensed time axis. Toggle it via Options → Show Timeline Strip. It is visible on the Gantt Chart and Team Planner tabs by default.
