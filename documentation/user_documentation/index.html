Step-by-step guides for every feature of the Project Offline application. Select a topic from the sidebar or the cards below.

What is Project Offline?

Project Offline is a desktop project-scheduling application that reads and writes Microsoft Project (.mpp / .xml) files via the MPXJ library. Its interface closely mirrors Microsoft Project, providing a Gantt chart, resource management, baseline tracking, dependency management, and team planning tools – all in a single window.

Note Files are saved as .mpp (Microsoft Project binary) or .xml (MSPDI format). Both formats are fully interoperable with Microsoft Project.

Manage team members, costs, and availability in the Resource Sheet.
🗓️

Team Planner

Drag tasks across resources and time on the interactive team timeline.
📈

Resource Usage Graph

Analyse workload allocation across monthly, weekly, daily and hourly views.
📊

Resource Usage Histogram

See total team workload per day at a glance — bars turn red when the team is over-allocated.
📌

Baseline Tracking

Capture a baseline snapshot and compare plan vs. actual schedule.
🔄

Confluence Calendar Sync

Automatically import public holidays and leave from Confluence Team Calendars.
📅

Holiday Calendar Management

Auto-populate project calendars with German national and state holidays or international public holidays. Import ICS feeds and add country calendars at any time.
📊

Critical Path Method (CPM)

Automatically identify critical tasks, analyse total and free float per task, and send the CPM report by email. Configure the critical slack threshold and dependency mode.
💾

Open, Save & Export

Open .mpp/.xml files, save, and export Gantt charts as SVG or PlantUML.
🗓️

Timeline Strip

Pin tasks and milestones to the executive Timeline strip above the views.
✂️

Task Splitting & Merging

Split tasks into non-contiguous segments and merge them back with full undo support.
🔐

KeePass Credentials

Open or create a KeePass database, configure Confluence SSO auto-fill, and use entries as Jira server credentials.
👥

Active Directory Sync

Look up employees by name, email, or username via PowerShell Get-ADUser and sync details into the resource sheet. Add new resources directly from AD with duplicate prevention and immediate ID assignment.
✉

Email Integration

Configure SMTP email delivery with KeePass credential storage to send project reports and Gantt chart exports.
🔗

Jira Integration

Connect to Jira Cloud or Server/Data Center with API Token, Password, or Personal Access Token authentication.
🔄

Jira Sync

Two-way sync between Jira and your project: import issues as tasks (Jira→Project) with epic hierarchy and undo support, and push tasks back to Jira (Project→Jira) with dry-run preview, conflict detection, and an audit trail.
🔌

Git & SVN Integration

Auto-commit on save, view history, restore revisions, manage branches, and sync with remote repositories using KeePass credentials.
⏳

Progress Dialogs

Long-running operations (file open, Jira sync, exports) run in background threads and display a modal progress dialog so the application stays responsive.
