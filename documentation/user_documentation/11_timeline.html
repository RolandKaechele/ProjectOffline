Timeline Strip

The compact timeline above the tab bar — an executive-friendly overview of the full project span, key tasks and milestones.

Home› Timeline Strip

What Is the Timeline Strip?

The Timeline Strip is a fixed-height bar displayed above the tab bar. It mirrors Microsoft Project's Timeline view: the full project span is shown on a condensed time axis with selected task bars and milestone diamonds pinned at their scheduled positions. It is ideal for executive summaries and slide exports.

Enabling / Disabling

Toggle via Options → Show Timeline Strip. The strip is visible by default on the Gantt Chart and Team Planner tabs only.

What Is Shown

ElementDescription
Light-blue background barFull project span (earliest start to latest finish)
Colour-cycled barsTasks that have been “registered” to the timeline (pinned)
Diamond shapesMilestones pinned to the timeline

Pinning Tasks to the Timeline

In the Gantt Chart or Task Sheet tab, right-click a task row and choose Show in Timeline to pin that task's bar or milestone diamond to the strip. Repeat to unpin (the context menu will show Remove from Timeline instead).

Scroll Sync

The timeline strip and the Gantt chart share the same horizontal scroll position and zoom level. When you zoom or scroll the Gantt, the timeline updates instantaneously. The strip's calendar header (month + day bands) stays in sync.

Layout

The strip is 178 px tall in total:

  • 42 px — two-row calendar header (month band + day band)
  • 136 px — canvas area for task bars and milestone diamonds

Multiple task bars are stacked in lanes (28 px each) to avoid overlap. Labels are centred inside each bar; milestone labels appear below the diamond.

Maximise / restore When the main window is maximised or restored the strip collapses to zero height briefly, then re-expands once Qt has settled the new layout. This prevents a flicker where the strip would render at the wrong width.
