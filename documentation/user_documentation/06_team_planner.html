Team Planner

See who is working on what and when — drag tasks between resources to reassign them without leaving the calendar view.

Home› Team Planner

What Is the Team Planner?

The Team Planner tab shows a resource-row timeline where each horizontal row represents one resource and each coloured bar represents a task assignment. Unlike the Gantt chart (which is task-centric), the Team Planner is resource-centric: you can immediately see who is busy, who is free, and whether anyone is over-allocated.

Reading the View

Visual elementMeaning
Blue task barAssigned task for this resource
Semi-transparent bar (ghost)Proposed position while dragging
Orange blockVacation / non-working period (from personal or Confluence calendar)
Yellow blockRegional public holiday from the resource’s secondary calendar (auto-assigned from AD location)
Green lineToday indicator
Grey columnSaturday / public holiday
Blue-tint columnSunday
Red row background + ! badgeOver-allocated resource (assigned > Max Units)

Unassigned Tasks Panel

The bottom strip shows task chips for tasks that have no resource assigned. Drag a chip from the unassigned panel onto a resource row to assign it and schedule it starting on the day where you drop it.

Drag-and-Drop Assignment

  1. Click and hold a task bar (or unassigned chip).
  2. Drag horizontally to reschedule, or vertically to a different resource row to reassign.
  3. A ghost bar follows the cursor; the target row highlights in blue.
  4. If the drop lands on a vacation block the ghost turns red — release to confirm anyway, or move the cursor to avoid it.
  5. Release the mouse to commit the change. The project is updated and the Gantt chart reflects the new dates.

Zoom & Navigation

The Team Planner shares the zoom level with the Gantt chart. Use the same zoom slider or the ◀ Today navigation button to scroll to the current date.

Scroll sync The calendar header and the resource rows scroll horizontally in perfect lock-step — a dedicated spacer in the layout ensures there is never a pixel drift between column headers and data cells regardless of the vertical scrollbar width.
Hover tooltips Hovering over an orange or yellow block shows a tooltip with the exception name and date range. For yellow (secondary calendar) blocks the tooltip also shows the name of the regional holiday calendar (e.g. Source: Bayern).
Secondary calendar date filtering Yellow blocks show only the exceptions that are unique to the secondary (regional) calendar — dates already present in the parent national calendar are filtered out to avoid double-rendering of national public holidays. The filter compares only named exceptions from the parent calendar; anonymous entries (e.g. unnamed school-holiday blocks from Confluence) are ignored, so a state holiday that falls on the same date as a Schulferien block is always displayed. When a secondary calendar exception has no name, the block label falls back to the calendar name (e.g. Baden-Württemberg); otherwise the label combines both as Calendar: Exception (e.g. Baden-Württemberg: Heilige Drei Könige).
All resources always visible The Team Planner displays a swimlane row for every named resource even if the resource's unique ID is unavailable or raises an error. Such resources appear with an empty task set and do not cause a crash. Newly added resources appear immediately because the application assigns a UID automatically on creation.
