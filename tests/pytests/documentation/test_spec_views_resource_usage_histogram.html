TestComputeHistogramDataSingleResource — Single resource capacity and allocation

RH-04Single resource capacity on working daysPASS
Init
One resource, no vacations; 5-day Mon–Fri window; 8 h/day stub.
Run
compute_histogram_data(project, mon, fri)
Expect
Each weekday entry has capacity_hours == 8.0.
Check
assert all(d["capacity_hours"] == 8.0 for d in result)
RH-05Single resource allocation distributed evenlyPASS
Init
One resource, one task spanning Mon–Fri with 40 h assignment.
Run
compute_histogram_data(project, mon, fri)
Expect
Each weekday has total_hours == 8.0 (40 h ÷ 5 working days).
Check
assert all(d["total_hours"] == 8.0 for d in result if d["date"].dayOfWeek() < 6)
RH-06Utilisation over 100 when overallocatedPASS
Init
One resource; task with 120 h assignment over a 5-day window (capacity 40 h total).
Run
compute_histogram_data(project, mon, fri)
Expect
At least one entry has utilisation_pct > 100.
Check
assert any(d["utilisation_pct"] > 100 for d in result)
RH-07Weekend days have zero capacityPASS
Init
One resource; 7-day Mon–Sun window including a weekend.
Run
compute_histogram_data(project, mon, sun)
Expect
Saturday and Sunday entries have capacity_hours == 0.
Check
assert all(d["capacity_hours"] == 0 for d in result if d["date"].dayOfWeek() >= 6)
