dialogs.py

105 tests
PyQt5 dialog classes: modal flags, title text, required controls and constructor signatures for all application dialogs. Includes CPM-based critical-path display in TaskDialog and a read-only Jira tab showing the linked issue key, status, and browse URL. Also covers the resource-assignment tab: 3-column QTableWidget (Name | Units % | ✕), per-row delete button via _del_btns list tracking, null-resource guard, and duplicate-resource prevention.
