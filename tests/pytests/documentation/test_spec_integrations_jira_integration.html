integrations/jira_integration.py

36 tests
Jira API connectivity and credential management for syncing tasks. Supports three authentication methods: API Token (Jira Cloud), Password, and Personal Access Token (Jira Server/Data Center). All JIRA connections are mocked — no real Jira server required. Covers test_connection with basic_auth and token_auth modes, get_jira_client creation, PAT/username validation, KeePass integration, config summaries, and credential whitespace stripping.
