integrations/ad_integration.py

63 tests
Active Directory lookup via PowerShell subprocess. All PS calls patched — no domain membership or RSAT required. Covers availability detection, lookup by name/email/username (with input sanitisation and injection-char rejection), lookup history recording, sync_resources, get_last_sync_result, and multi-result lookups (lookup_by_email_all, lookup_by_username_all) including the fast-path exact-match optimisation and wildcard fallback. Also covers search_groups() (name-wildcard AD group search: empty/injection/single-quote guards, single vs. array JSON parsing, schema validation, group filtering) and get_group_members() (enabled-user member enumeration: injection guards, email validation, recursive flag, single/multi-user parsing).
