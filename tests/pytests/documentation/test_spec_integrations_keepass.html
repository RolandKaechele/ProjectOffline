integrations/keepass_integration.py

52 tests
KeePassManager owns the in-memory pykeepass session. Tests cover singleton lifecycle, state queries, unlock/lock/auto-unlock (incl. failure path and key-file forwarding), database creation (absolute path conversion, parent-dir creation, keyfile persistence, failure), entry CRUD (locked and unlocked), save exception path, module-level convenience functions, and key-file generation (64 binary bytes, randomness, binary format guard, bad-path error).
