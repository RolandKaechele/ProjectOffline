ribbon.py

63 tests
Ribbon widget: tab/panel/group/button construction, set_save/project_open/delete/confluence state, tab-changed signal. Includes add_menu_button() helper and split-button action-count tests for all converted dropdown buttons (Jira Sync/Push, Add Resource, Gantt/Timeline/Resource-SVG Export, Set/Clear Baseline, VCS Commit/Pull/SVN Update). Email-disabled state: _RibbonListPopup.set_item_enabled() (disable/enable/unknown-label no-op) and ProjectRibbon.set_email_actions_enabled() (disables/re-enables all three email popup items across all three export popups).
