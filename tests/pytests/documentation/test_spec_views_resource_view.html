resource_view.py

56 tests
Resource sheet view: construction, load/clear, resource display, delete selected, data-changed signal. Also covers _ensure_resource_uid(): skips when UID already set, assigns next free UID when None, avoids collisions with existing UIDs, starts at 1 for an empty project. Covers the updated add_resource(): ID assigned and personal calendar created before dialog opens; calendar and resource rolled back on Cancel. Covers add_resources_from_ad_group(): group-member bulk import via ADGroupSearchDialog, duplicate detection, email assignment, data_changed emission. Covers _create_resource_calendar(): links personal calendar to project standard calendar via setParent(); falls back to addDefaultCalendarDays() when no default; UID collision avoidance.
