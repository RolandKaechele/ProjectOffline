team_planner_view.py

Team Planner: chip duration string, lane layout, row height, canvas/view construction, placement dialog, vacation blocks, overlap detection, conflict data, signals, segment vacation overlap. Also covers resource filter and UID handling in load_project(): null-UID resources appear as empty rows, resources with null UID are excluded from _tasks_by_res, arbitrary UID exceptions are caught silently, resources with valid UIDs are keyed correctly.
